Details about Swift Airline Ticket Booking Project

Swift is a Dummy Airline Booking Website without payment gateway.
Explore features of Swift Airline Ticket Booking Website like Redeem Code, User Booking Cancellation based on Journey accomplishment and many more.

Swift has provision Admin Pannel for Admin user for Backend which provide details about Booking Details of all Users and has provision of cancel booking, also all Airline Details including Flights routes, Departure City & Time, Arrival City & Time and more which has provision for Admin to Create Flight Route, Update existing, and Delete.
User must Login as Admin to enable Backend
